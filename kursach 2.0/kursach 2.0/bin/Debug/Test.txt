1. Какой оператор цикла используется для выполнения блока кода определенное количество раз?
   a) Цикл for
   b) Цикл while
   c) Цикл do-while
Ответ: a.
2. Когда условие цикла проверяется в цикле while?
   a) Перед каждой итерацией
   b) После каждой итерации
   c) Только один раз перед началом цикла
Ответ: a.
3. В чем отличие цикла do-while от цикла while?
   a) Цикл do-while выполняется хотя бы один раз
   b) Цикл do-while не имеет условия завершения
   c) Цикл do-while используется для перебора массивов
Ответ: a.
4. Что такое бесконечный цикл?
   a) Цикл, который никогда не завершается
   b) Цикл, который выполняется только один раз
   c) Цикл, который выполняется определенное количество раз
Ответ: a.
5. Какой оператор позволяет прервать выполнение цикла?
   a) break
   b) continue
   c) skip
Ответ: a.
6. Для чего используются вложенные циклы?
   a) Для выполнения нескольких операций одновременно
   b) Для пропуска блоков кода
   c) Для создания бесконечных циклов
Ответ: a.
7. Что позволяют делать операторы управления циклом?
   a) Пропускать блоки кода или переходить к определенной части цикла
   b) Увеличивать количество итераций цикла
   c) Завершать выполнение программы
Ответ: a.
8. Какой цикл используется, когда количество итераций заранее не известно?
   a) Цикл for
   b) Цикл while
   c) Цикл do-while
Ответ: b.
9. Какой оператор позволяет пропустить оставшуюся часть текущей итерации и перейти к следующей итерации цикла?
   a) break
   b) continue
   c) skip
Ответ: b.
10. Для чего обычно используются вложенные циклы?
   a) Для обработки многомерных массивов данных или для выполнения нескольких операций одновременно
   b) Для создания бесконечных циклов
   c) Для прерывания выполнения программы
Ответ: a.
